# L-Evolution-p-dagogique-en-France
《教育思想的演进》读书笔记。

从这本书中能学到三件事。第一，在历史学方面，一个人能了解西欧从加洛林时代到二十世纪初的一千余年的教育演进，包括大教育家及其教育思想、教育教学机构和教学形式、还有教学内容的演进。在探索这个演化的过程中，我们能很清楚地看到现代教育体系的源流。

第二，在社会学方面，这本书作为涂尔干的代表作，运用实证主义精神的社会学方法论，把教育作为一个丰富的实例进行研究，研究在各种社会条件下，教育会呈现怎样的面貌，发生怎样的变化。这是一个非常典型和成熟的早期社会学研究例子。

第三，在教育学方面，通过研究其演进过程，我们也能获得对当下情况的启发，思考教育学未来应当走的方向。这是这项研究的现实意义。
